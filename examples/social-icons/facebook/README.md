# facebook

An animated Facebook logo (msvg package).

- **intro:** the mark fades and scales in with a soft overshoot.
- **loadingLoop:** a calm, confident breath — `scale(1)` -> `1.06` -> `1`, then a short rest, looping seamlessly (first and last keyframes both `scale(1)`).
- **hover:** a single confident pulse.
