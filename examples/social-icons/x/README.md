# x

An animated X logo (msvg package).

- **intro:** the mark snaps in crisply from a slight scale and counter-rotation (~420ms).
- **loadingLoop:** a sharp minimal presence — a slow scale breath to `1.06` with one crisp `rotate(4deg)` tick, ending exactly at `rotate(0) scale(1)`, looping seamlessly (first and last keyframes identical).
- **hover:** a quick crisp rotate tick.
