# tiktok

An animated TikTok logo (msvg package).

- **intro:** the mark pops in with a soft overshoot (~460ms).
- **loadingLoop:** a musical bass line — `scale(1)` -> `1.08` -> `1` -> `1.05` -> `1` double-thump, then a rest, looping seamlessly (first and last keyframes both `scale(1)`).
- **hover:** a single beat thump.
