# help-outline

An animated msvg package for the Material Symbols **help_outline** icon.

- **Loop concept:** the question mark tilts curiously to +7deg, holds a beat, and returns — inviting you to ask.
- **Timelines:** `intro` (fade + scale in), `loadingLoop` (curious tilt + rest), `hover` (single tilt).

Icon from Google Material Symbols (Apache-2.0).
