# face

An animated msvg package for the Material Symbols **face** icon.

- **Loop concept:** the two eyes blink together — scaleY 1 to 0.12 and back in about 200ms — once per cycle, a friendly presence.
- **Timelines:** `intro` (fade + scale in), `loadingLoop` (synchronized blink + rest), `hover` (single blink).

Icon from Google Material Symbols (Apache-2.0).
