# groups

An animated msvg package for the Material Symbols **groups** icon.

- **Loop concept:** one shared breath — the central figure scales gently to 1.04 and back while the team stays present around it.
- **Timelines:** `intro` (fade + settle), `loadingLoop` (central breath + rest), `hover` (single breath).

Icon from Google Material Symbols (Apache-2.0).
