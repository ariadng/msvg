# file-download

An animated msvg package for the Material Symbols **file_download** icon.

- **Loop concept:** the arrow descends into the tray and fades out, then teleports back to the top and fades in — a repeating download.
- **Timelines:** `intro` (fade + drop in), `loadingLoop` (arrow descent + fade + reset), `hover` (small dip).

Icon from Google Material Symbols (Apache-2.0).
