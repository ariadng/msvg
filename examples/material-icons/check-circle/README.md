# check-circle

An animated msvg package for the Material Symbols **check-circle** icon.

- **Loop concept:** one verified beat — a whole-glyph pulse to 1.06 with a fast attack and slow decay, then a long rest.
- **Timelines:** `intro` (pop in), `hover` (single beat), `loadingLoop` (seamless verified beat).

Icon from Google Material Symbols (Apache-2.0).
