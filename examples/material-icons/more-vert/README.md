# more-vert

An animated msvg package for the Material Symbols **more_vert** icon.

- **Loop concept:** the three dots bounce in sequence — each lifts 25 units with a 120ms stagger, then rest — the universal typing indicator.
- **Timelines:** `intro` (fade + scale in), `loadingLoop` (staggered dot bounce + rest), `hover` (single sequence).

Icon from Google Material Symbols (Apache-2.0).
