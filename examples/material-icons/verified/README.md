# verified

An animated msvg package for the Material Symbols **verified** icon.

- **Loop concept:** one proud stamp-pulse — the badge scales to 1.06 with a +3deg flick, settles, and rests — an authenticity gleam.
- **Timelines:** `intro` (fade + scale + rotate in), `loadingLoop` (stamp-pulse + rest), `hover` (single pulse).

Icon from Google Material Symbols (Apache-2.0).
