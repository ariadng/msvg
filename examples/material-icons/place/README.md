# place

An animated msvg package for the Material Symbols **place** icon.

- **Loop concept:** a gentle hover-float — the pin drifts 22 units up and slowly settles — pinned here, distinct from a hop.
- **Timelines:** `intro` (fade + drop in), `loadingLoop` (slow float + rest), `hover` (single float).

Icon from Google Material Symbols (Apache-2.0).
