# location-on

An animated msvg package for the Material Symbols **location-on** icon.

- **Loop concept:** you are here — the pin hops up 30 units and lands with a small squash at its tip (origin bottom-center), then rests planted.
- **Timelines:** `intro` (drop in), `hover` (hop), `loadingLoop` (seamless hop + tip squash).

Icon from Google Material Symbols (Apache-2.0).
