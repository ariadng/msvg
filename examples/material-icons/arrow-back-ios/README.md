# arrow-back-ios

An animated msvg package for the Material Symbols **arrow_back_ios** icon.

- **Loop concept:** a leftward beckon — the chevron nudges 40 units left and softly returns — the mirror of arrow-forward-ios, inviting you back.
- **Timelines:** `intro` (fade + scale in), `loadingLoop` (leftward nudge + rest), `hover` (single nudge).

Icon from Google Material Symbols (Apache-2.0).
