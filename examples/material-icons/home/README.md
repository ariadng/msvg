# home

An animated msvg package for the Material Symbols **home** icon.

- **Loop concept:** a gentle float — the house rises about 25 user units and softly settles back, using soft easing for a calm, buoyant idle.
- **Timelines:** `intro` (fade + rise in), `hover` (small lift), `loadingLoop` (seamless float).

Icon from Google Material Symbols (Apache-2.0).
