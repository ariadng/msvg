# manage-accounts

An animated msvg package for the Material Symbols **manage_accounts** icon.

- **Loop concept:** the gear ratchets forward in two crisp 15deg steps then sweeps back while the person stays calm — settings applied to the account.
- **Timelines:** `intro` (fade + scale in), `loadingLoop` (gear ratchet + rest), `hover` (single turn).

Icon from Google Material Symbols (Apache-2.0).
