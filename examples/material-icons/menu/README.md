# menu

An animated msvg package for the Material Symbols **menu** icon.

- **Loop concept:** a soft breathing pulse — the three lines scale up to ~1.07 and back with a faint opacity dip, giving a calm inhale/exhale rhythm.
- **Timelines:** `intro` (fade + drop in), `hover` (small scale), `loadingLoop` (seamless breathing pulse).

Icon from Google Material Symbols (Apache-2.0).
