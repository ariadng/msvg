# search

An animated msvg package for the Material Symbols **search** icon.

- **Loop concept:** a lens sweep — the magnifier drifts through a small circular path (multi-keyframe translate) at a steady pace, like scanning an area.
- **Timelines:** `intro` (fade + settle from a slight tilt), `hover` (subtle zoom), `loadingLoop` (seamless circular drift).

Icon from Google Material Symbols (Apache-2.0).
