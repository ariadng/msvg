# cancel

An animated msvg package for the Material Symbols **cancel** icon.

- **Loop concept:** a firm head-shake no — rotate -8/+8/-3 and settle — a decisive cancelling gesture, then a long rest.
- **Timelines:** `intro` (fade + scale in), `loadingLoop` (head-shake + rest), `hover` (single shake).

Icon from Google Material Symbols (Apache-2.0).
