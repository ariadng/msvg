# description

An animated msvg package for the Material Symbols **description** icon.

- **Loop concept:** content loading — the two text lines pulse top-to-bottom in sequence, as if text is appearing on the page.
- **Timelines:** `intro` (fade + settle), `hover` (line sweep), `loadingLoop` (seamless top-to-bottom text pulse).

Icon from Google Material Symbols (Apache-2.0).
