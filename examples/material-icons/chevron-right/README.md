# chevron-right

An animated msvg package for the Material Symbols **chevron-right** icon.

- **Loop concept:** proceed — a rightward beckon, the chevron nudges 45 units right and eases back, mirroring the back arrow.
- **Timelines:** `intro` (slide in), `hover` (single beckon), `loadingLoop` (seamless rightward beckon).

Icon from Google Material Symbols (Apache-2.0).
