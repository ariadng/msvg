# language

An animated msvg package for the Material Symbols **language** icon.

- **Loop concept:** the globe breathes once — a slow soft scale to 1.05 and a gentle decay back to rest — a calm worldwide presence.
- **Timelines:** `intro` (fade + scale in), `loadingLoop` (single breath + rest), `hover` (single breath).

Icon from Google Material Symbols (Apache-2.0).
