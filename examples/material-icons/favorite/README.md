# favorite

An animated favorite heart icon (msvg package).

- **Intro:** the heart pops in with a soft overshoot.
- **loadingLoop:** a heartbeat — `scale(1)` -> `1.15` -> `1` -> `1.08` -> `1` double-thump, then a short rest, looping seamlessly (first and last keyframes both `scale(1)`).
- **hover:** a single affectionate beat.

Icon from Google Material Symbols (Apache-2.0).
