# visibility

An animated msvg package for the Material Symbols **visibility** icon.

- **Loop concept:** a real blink — the eye squashes shut (scaleY to 0.12) and reopens, then the iris glances left-right during the long open rest.
- **Timelines:** `intro` (fade + settle), `hover` (blink), `loadingLoop` (seamless blink + glance).

Icon from Google Material Symbols (Apache-2.0).
