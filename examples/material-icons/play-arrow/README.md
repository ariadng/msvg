# play-arrow

An animated msvg package for the Material Symbols **play_arrow** icon.

- **Loop concept:** a press gesture — the triangle scales to 0.94, holds, then releases with a 1.03 overshoot and settles — inviting the tap.
- **Timelines:** `intro` (fade + scale in), `loadingLoop` (press + release + rest), `hover` (single press).

Icon from Google Material Symbols (Apache-2.0).
