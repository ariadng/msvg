# settings

An animated settings gear icon (msvg package).

- **Intro:** the gear fades and rotates into its resting position.
- **loadingLoop:** the canonical loading spinner — a seamless, linear 360deg spin (`rotate(0deg)` -> `rotate(360deg)`) that repeats while work is in progress.
- **hover:** a small responsive nudge and back.

Icon from Google Material Symbols (Apache-2.0).
