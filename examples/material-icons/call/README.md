# call

An animated msvg package for the Material Symbols **call** icon.

- **Loop concept:** one soft ring burst — four quick +/-4deg rocks, then a long silence, like a classic incoming ring.
- **Timelines:** `intro` (fade + scale in), `loadingLoop` (ring burst + silence), `hover` (single decaying ring).

Icon from Google Material Symbols (Apache-2.0).
