# schedule

An animated msvg package for the Material Symbols **schedule** icon.

- **Loop concept:** the clock hands tick forward in two crisp 15deg steps then sweep back to twelve, marking time passing.
- **Timelines:** `intro` (fade + scale in), `loadingLoop` (two-step tick + sweep back), `hover` (single tick).

Icon from Google Material Symbols (Apache-2.0).
