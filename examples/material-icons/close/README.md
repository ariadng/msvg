# close

An animated msvg package for the Material Symbols **close** icon.

- **Loop concept:** a slow tick — the X rotates a quarter turn (90deg) and back, holding briefly at each end so it reads like a patient clock tick rather than a spin.
- **Timelines:** `intro` (fade + unwind from -90deg), `hover` (quarter-turn nudge), `loadingLoop` (seamless tick with pauses).

Icon from Google Material Symbols (Apache-2.0).
