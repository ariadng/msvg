# lock

An animated msvg package for the Material Symbols **lock** icon.

- **Loop concept:** secured — two tiny ±3° shakes, like testing the shackle, then the lock holds firmly still for most of the cycle.
- **Timelines:** `intro` (fade + settle), `hover` (test wiggle), `loadingLoop` (seamless locked-tight wiggle).

Icon from Google Material Symbols (Apache-2.0).
