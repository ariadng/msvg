# arrow-back

An animated msvg package for the Material Symbols **arrow-back** icon.

- **Loop concept:** go back — one leftward beckon, the arrow slides 45 units left with a soft return, then rests.
- **Timelines:** `intro` (slide in), `hover` (single beckon), `loadingLoop` (seamless leftward beckon).

Icon from Google Material Symbols (Apache-2.0).
