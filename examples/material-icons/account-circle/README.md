# account-circle

An animated msvg package for the Material Symbols **account-circle** icon.

- **Loop concept:** a single calm presence pulse — the avatar breathes (scale to ~1.03 with a faint opacity dip) then rests.
- **Timelines:** `intro` (fade + settle), `hover` (soft pulse), `loadingLoop` (seamless presence breath).

Icon from Google Material Symbols (Apache-2.0).
