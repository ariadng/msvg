# local-shipping

An animated msvg package for the Material Symbols **local-shipping** icon.

- **Loop concept:** delivery en route — the truck road-jiggles, bouncing twice on its suspension (the wheels are counter-wound holes in the body, so the whole silhouette bounces together), then rests.
- **Timelines:** `intro` (roll in), `hover` (single bounce), `loadingLoop` (seamless road-jiggle).

Icon from Google Material Symbols (Apache-2.0).
