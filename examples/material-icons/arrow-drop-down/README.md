# arrow-drop-down

An animated msvg package for the Material Symbols **arrow_drop_down** icon.

- **Loop concept:** a small downward dip — the caret nods 30 units down and springs softly back — pointing to options below.
- **Timelines:** `intro` (fade + drop in), `loadingLoop` (downward dip + rest), `hover` (single dip).

Icon from Google Material Symbols (Apache-2.0).
