# email

An animated msvg package for the Material Symbols **email** icon.

- **Loop concept:** new mail lands — the envelope lifts, drops back, and settles with a soft landing squash (scaleY to 0.97).
- **Timelines:** `intro` (drop in), `hover` (drop-settle), `loadingLoop` (seamless mail-lands drop).

Icon from Google Material Symbols (Apache-2.0).
