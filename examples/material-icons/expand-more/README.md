# expand-more

An animated msvg package for the Material Symbols **expand-more** icon.

- **Loop concept:** one beckoning nod — the chevron dips down 45 units and eases back, inviting “more below.”
- **Timelines:** `intro` (drop in), `hover` (single dip), `loadingLoop` (seamless downward nod).

Icon from Google Material Symbols (Apache-2.0).
