# favorite-border

An animated msvg package for the Material Symbols **favorite-border** icon.

- **Loop concept:** wishing — a single soft flutter (1 → 1.08 → 1 → 1.04 → 1), gentler and with a longer rest than the filled heart.
- **Timelines:** `intro` (bloom in), `hover` (flutter), `loadingLoop` (seamless soft flutter).

Icon from Google Material Symbols (Apache-2.0).
