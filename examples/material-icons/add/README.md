# add

An animated add (plus) icon (msvg package).

- **Intro:** the plus rotates and scales into place.
- **loadingLoop:** a pulse plus quarter-turn — `rotate(0deg)` -> `rotate(90deg)` with a scale breath. Because a plus is 90deg-rotationally symmetric, the first and last keyframes look identical, so the loop is seamless.
- **hover:** a crisp quarter-turn snap.

Icon from Google Material Symbols (Apache-2.0).
