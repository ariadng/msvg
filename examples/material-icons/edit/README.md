# edit

An animated msvg package for the Material Symbols **edit** icon.

- **Loop concept:** the pencil scribbles — three small diagonal strokes along its writing direction with a slight tip-down tilt, then rest.
- **Timelines:** `intro` (fade + settle), `hover` (single stroke), `loadingLoop` (seamless three-stroke scribble).

Icon from Google Material Symbols (Apache-2.0).
