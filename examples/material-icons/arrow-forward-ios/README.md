# arrow-forward-ios

An animated msvg package for the Material Symbols **arrow_forward_ios** icon.

- **Loop concept:** a rightward beckon — the chevron nudges 40 units right and softly returns — inviting you to proceed.
- **Timelines:** `intro` (fade + slide in), `loadingLoop` (rightward nudge + rest), `hover` (single nudge).

Icon from Google Material Symbols (Apache-2.0).
