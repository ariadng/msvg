# done

An animated msvg package for the Material Symbols **done** icon.

- **Loop concept:** the check stamps once — a quick compress to 0.92 then an emphasized snap to 1.06 and settle, like a confident tap.
- **Timelines:** `intro` (pop in), `hover` (stamp), `loadingLoop` (seamless confirm stamp).

Icon from Google Material Symbols (Apache-2.0).
