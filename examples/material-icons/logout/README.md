# logout

An animated msvg package for the Material Symbols **logout** icon.

- **Loop concept:** signing out — the arrow nudges rightward out of the door frame 45 units and returns, while the frame stays put.
- **Timelines:** `intro` (fade + settle), `hover` (arrow nudge), `loadingLoop` (seamless step-out nudge).

Icon from Google Material Symbols (Apache-2.0).
