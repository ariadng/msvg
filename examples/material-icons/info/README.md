# info

An animated msvg package for the Material Symbols **info** icon.

- **Loop concept:** the i-dot pops (scale to 1.4 and back) to whisper “psst, information here” while the ringed body stays still.
- **Timelines:** `intro` (fade + settle), `hover` (dot pop), `loadingLoop` (seamless dot pop with a long rest).

Icon from Google Material Symbols (Apache-2.0).
